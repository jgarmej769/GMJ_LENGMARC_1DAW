AÑADIR IMÁGENES

De igual forma que podemos introducir texto, mediante etiquetas, podemos insertar imágenes de la forma que necesitemos. La principal etiqueta usada es <img>. Dicha etiqueta está compuesta por diversas partes, las cuales detallaremos en la siguiente tabla;

ATRIBUTO DESCRIPCIÓN
src Se coloca la ruta o URL de la imagen a enlazar.
alt Es el texto alternativo que saldrá en el caso de que la imagen no cargue.
width / height Ajusta el tamaño de la imagen. en píxeles.

AÑADIR ENLACES

Para colocar enlaces, existe la etiqueta <a>. Con dicha etiqueta, colocamos un texto al cual se le añade un hipervínculo dado, los atributos de dicho texto vienen dado por;

ATRIBUTO DESCRIPCIÓN
href La URL de destino del enlace. Obligatorio si quieres que funcione como enlace.
>[texto]< Texto que aparecerá para hacer click.

También podemos modiifcar como se ve nuestro enlace, como se ve antes de visitarlo, después de haberlo visitado, y mientras se le pasa el cursor por encima.
Para ello; debemos crear una "pseudoclase", donde estableceremos cómo se verán los enlaces de nuestra web.
Esta pseudoclase puede ir acompañada por un "style" o similar, que modifique color, tamaño, o fuente.

PSEUDOCLASE ESTADO DEL ENLACE
:link El enlace se encuentra en estado normal, es decir; no visitado.
:visited El enlace ya ha sido visitado.
:hover En cursor se encuentra sobre el enlace, epro sin hacer click.
:active El enlace está activo, es decir, se está haciendo click sobre él.

FORMULARIOS

Para tener una buena web, se puede precisar de un formulario, en el cual se soliciten datos.
Se utiliza la etiqueta <form>, dentro de ella, podemos usar diversos atributos, así como podemos modificar su tipo.
Se pueden cambiar su principal atributo <type>,tal que:

TIPO DESCRIPCIÓN
radio Será undesplegable, dónde podemos seleccionar una opción
checkbox En esta ocasión, las opciones vienen marcadas por un botón que pulsar.
email Ahora tendremos un campo donde poder colocar un correo electrónico.
tel Campo para colocar un número de teléfono, con su formato.
password Es un campo de texto oculto, es decir, no se ve nada al introducir texto.
file Permite subir un archivo o similar.
reset Botón que limpia los campos anteriores.

TABLAS

Como viene siendo obvio, en una página web viene a ser casi imprescindible. Para realizar una tabla usaremos la etiqueta < table >, la cual se puede modificar con las siguientes etiquetas;

ETIQUETA DESCRIPCIÓN
< tr > Crea una nueva fila, dónde podremos colocar texto.
< th > Sirve para crear una nueva columna, usando esta etiqueta, el texto se diferencia de los que se encuentran dentro de las demás celdas.
< td > Sirve para crear otras celdas, que le introduciremos texto.

Dentro de cada etiqueta, podemos colocar el texto que deseemos. Dentro del < th > que establece las columnas, debemos saber que; a más < th > colocamos, más columnas se "crean".
